# Compose Server

快速且对开发者友好的服务端快速开发框架，涵盖了大多开发场景，未来可期。
其采用[多项目模块化设计](/documentation/model_manifest.md)，
可[不同环境进行测试](/documentation/build_env.md)


> 警告，此项目目前暂时采用私有协议，仅公开源代码，详见 [LICENSE](/LICENSE)
